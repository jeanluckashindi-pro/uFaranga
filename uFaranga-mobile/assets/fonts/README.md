# Polices du projet uFaranga

Ce dossier contient les fichiers de polices Google Fonts utilisés dans le projet.

## 📥 Téléchargement des polices

Téléchargez les polices suivantes depuis Google Fonts et placez les fichiers `.ttf` dans ce dossier :

### Polices principales
- **Open Sans** : https://fonts.google.com/specimen/Open+Sans
  - `OpenSans-Regular.ttf`
  - `OpenSans-Bold.ttf`

- **Josefin Sans** : https://fonts.google.com/specimen/Josefin+Sans
  - `JosefinSans-Regular.ttf`
  - `JosefinSans-Bold.ttf`

### Polices supplémentaires
- **Anton** : https://fonts.google.com/specimen/Anton
  - `Anton-Regular.ttf`

- **Antonio** : https://fonts.google.com/specimen/Antonio
  - `Antonio-Regular.ttf`

- **Bangers** : https://fonts.google.com/specimen/Bangers
  - `Bangers-Regular.ttf`

- **Cookie** : https://fonts.google.com/specimen/Cookie
  - `Cookie-Regular.ttf`

- **Allan** : https://fonts.google.com/specimen/Allan
  - `Allan-Regular.ttf`

- **Luckiest Guy** : https://fonts.google.com/specimen/Luckiest+Guy
  - `LuckiestGuy-Regular.ttf`

- **Ubuntu** : https://fonts.google.com/specimen/Ubuntu
  - `Ubuntu-Regular.ttf`

## 🚀 Installation rapide

Vous pouvez télécharger toutes les polices en une fois :
1. Visitez chaque lien ci-dessus
2. Cliquez sur "Download family"
3. Extrayez les fichiers `.ttf` nécessaires
4. Placez-les dans ce dossier `assets/fonts/`

## 📝 Note
Les polices sont automatiquement chargées via le hook `useProjectFonts()` dans `hooks/use-fonts.ts`.
